# SortingAlgosReact
Website written in React with graphic represantation of sorting algorithms.

Live Demo - 

List of sorting algorithms:

- Bubble Sort;
- Merge Sort;
- Insertion Sort;
- Selection Sort;
- Shell Sort;
- Heap Sort;
- Quick Sort;


How to deploy on your own machine:

1. npm install
2. npm start
